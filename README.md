# CSS-Variable-Image-Controller

A simple and interactive project that demonstrates how to control CSS variables dynamically using JavaScript.

This project allows users to:
- Adjust image padding (spacing)
- Apply real-time blur effects
- Change the base background color of the image

All updates happen live using CSS custom properties (`--variables`) controlled by JavaScript.
